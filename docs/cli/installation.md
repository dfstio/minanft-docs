# Installation

You need to install node and git
and clone the repo

```

    git clone https://github.com/dfstio/minanft-cli
    cd minanft-cli
    yarn
```

Make sure that minanft command is executable by running from the minanft-cli folder

```
    chmod +x ./src/cli.ts
    npm link
```
