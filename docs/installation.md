# Installation

## Add minanft to you repo

```
yarn add minanft
```

## Import for TypeScript

```typescript
import { MinaNFT, MinaNFTNameService, MINANFT_NAME_SERVICE } from "minanft";
```

## Import for JavaScript

```javascript
const {
  MinaNFT,
  MinaNFTNameService,
  MINANFT_NAME_SERVICE,
} = require("minanft");
```
